For the third part, you should try to click numbers from 1 to 9. It would change the shape that is rotating to the polygon of this number of sides (if 1, then a point. if 2, then a line). Also, if you click any button, the rotating shape will change its color to a random one. 

When starting the program, the color may end up being dark so its hard to see the shape. You could restart the proram or click any button to change the color.

Also, I modified the makefile to include the math library when compiling the funanim.c, so please use my makefile.
